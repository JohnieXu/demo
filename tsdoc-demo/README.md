# tsdoc-demo

这是一个采用 **TSDoc + Vuepress@1.x** 方案自动生成开源（私有）项目文档的 demo，详细介绍可查看这篇文章：[https://juejin.cn/post/7275943600780787753](https://juejin.cn/post/7275943600780787753)

## 使用

安装依赖

```bash
npm i
```

启动 Vuepress 文档本地预览

```bash
npm run docs:dev
```

使用 Vuepress 构建文档静态资源

```bash
npm run docs:build
```
